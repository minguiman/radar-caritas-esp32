Uso rapido

1. Pulsa Importar carpeta y elige faces para cargar los .anim actuales, o una carpeta con imagenes.

2. Selecciona una animacion, arrastra frames para reordenar, o usa insertar/eliminar.

3. Pulsa Generar carpeta faces. Elige una carpeta de salida o la raiz de la SD.

Tambien puedes arrastrar imagenes aqui para anadirlas a la animacion seleccionada.

Clasificacion

La carpeta exportada sale como faces/categoria/animacion.anim. Usa estas categorias si quieres que el firmware las encuentre en sus eventos actuales.

Si exportas a la SD, deja solo faces. El firmware actual no necesita JPG, PNG, RGB565 ni manifest en la SD.
